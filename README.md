## Install

- Backend:
  - FastAPI:

    `virtualenv .venv`

    `source .venv/bin/activate`

    `pip install "fastapi[standard]"`

    `pip install numpy`

- Frontend:
  - next:
  
    `npm i`

## Run

- Backend:
  - FastAPI:

    `fastapi run api.py`

- Frontend:
  - next:
  
    `npm run dev`

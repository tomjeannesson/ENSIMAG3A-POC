# Projet POC

All documentation is available here: [https://github.com/tomjeannesson/ENSIMAG3A-POC/wiki](https://github.com/tomjeannesson/ENSIMAG3A-POC/wiki)

## Install

- Backend:
  - FastAPI:

    `virtualenv .venv`

    `source .venv/bin/activate`

    `pip install "fastapi[standard]"`

    `pip install numpy`

- Frontend:
  - next:
  
    `npm i`

## Run

- Backend:
  - FastAPI:

    `fastapi run api.py`

- Frontend:
  - next:
  
    `npm run dev`
